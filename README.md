Login4tests
===========

A very basic login app - without database - built on a bootstrap 3 template. 

Clone repo
```
 $ git clone https://github.com/VerAx1/login4tests
```
Run bundle install
```
 $ bundle
```
Run with:
```
 $ ruby web.rb
```

### Requirements

- ruby 2.0.0-p247
- bundler
